# Support / 支援

## 使用問題與一般錯誤

請先確認瀏覽器在支援範圍內，重新載入 App，並嘗試匯出備份。若問題持續，請到
[GitHub Issues](https://github.com/fuyo1622/acg_app/issues) 建立 issue，附上：

- 裝置、作業系統、瀏覽器名稱與版本
- 重現步驟及預期／實際結果
- 可公開的錯誤畫面或主控台訊息

請勿上傳包含私人收藏、備註或照片的 JSON 備份。這個 App 沒有伺服器端資料副本，
沒有備份時，維護者無法復原瀏覽器內被清除的資料。

## Usage questions and ordinary bugs

Confirm that the browser is in the supported range, reload the app, and try to export a
backup. If the problem continues, open a
[GitHub issue](https://github.com/fuyo1622/acg_app/issues) with:

- Device, operating system, browser name, and browser version
- Reproduction steps and expected/actual behavior
- Screenshots or console messages that are safe to share publicly

Do not attach JSON backups containing private collection data, notes, or photos. This app
has no server-side copy, so maintainers cannot recover browser data that was cleared
without a backup.

## Security reports

Do not report vulnerabilities in a public issue. Follow [SECURITY.md](SECURITY.md).
